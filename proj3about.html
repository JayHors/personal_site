About this Project

This app allows you to browse cards throughout the history of the Pokemon Trading Card Game. You will be able to save future cards, too!

It makes use of the Pokemon TCG API, avaiable here. It grabs the sets from there, as well as using a driver for Dart found in the documentation.

I am proud of my robust usage of sharedPreferences here to both store the last set searched, as well as storing the collection.

I wish I could properly search the API, but by the time I had gotten to the point where I was looking to granulate my search, I discovered that the Dart driver doesn't currently allow for it. (If you look into my search page widget, you can even see that I had it mostly implemented!)

While it isn't reviewed and merged in time for the submission deadline, I did start work on updating the Dart driver to allow for this type of thing here.

It was also far too along in the process for me to rewrite using the REST API, as a lot of the widget building is dependent on the PokemonCard class provided by the driver.

I really feel that the app should have the ability to search cards by name, but like I said, it's mostly implemented, just removed for the time being.
