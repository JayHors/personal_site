ThoughtMap

ThoughtMap allows users to map out their thoughts, literally! Uses React and MapBox for the front end, and Node.js and Express.js for the backend. Has a basic monetization system built in, in the form of an ad network (fake).

SimplyPlace

SimplyPlace is a site that lets users find places near them and save them for later. Uses the Google Maps Places API for places and Firebase for syncing across clients, along with Bulma for CSS. Vanilla JS otherwise!
